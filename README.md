Aho-Sweeper
====================================================================================================
これと言って特徴のない超シンプルなデスクトップLinux向けマインスイーパーアプリ。
けっこうな頻度で解けないパターンが出るのでアホです。

![画像](etc/screenshot-1.png)

インストール方法
----------------------------------------------------------------------------------------------------
### ビルド
コンパイルするには以下のパッケージのインストールが必要です。

* gtk3
* meson
* valac

コンパイル手順は以下になります。


1. プロジェクトのルートディレクトリで作業をします。

       $ meson --prefix=/usr/local build
	   $ cd build
	   $ ninja

2. 以下のコマンドでインストールします。

       $ sudo ninja install
